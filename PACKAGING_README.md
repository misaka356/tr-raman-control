# Packaging Notes

## Goal

Create a distributable folder that includes:

- packaged executable
- startup batch file
- config template
- documentation

## End User Workflow

1. unpack the folder
2. edit `app_config.json`
3. double-click `start_tr_raman.bat`

## Files

- `start_tr_raman.bat`
- `app_config.template.json`
- `app_config.json`
- `TRRamanIntegratedController\TRRamanIntegratedController.exe`
- `README.txt`
